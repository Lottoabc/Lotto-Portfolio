Lotto's XR 开发作品集

欢迎来到我的 XR 开发世界！这里展示了我近期在虚拟现实 (VR) 和增强现实 (AR) 领域的一些实践项目，旨在探索沉浸式体验的无限可能。由于视频资源托管于 YouTube，部分地区可能需要网络代理才能流畅观看。

案例 1: 基于 PicoSDK 和 XRToolkit 的交互开发

  • 凝视交互：通过精准的视线追踪实现直观高效的交互逻辑。
  • 手势识别与追踪：结合先进的手部追踪技术，让用户能以自然手势操控虚拟环境。
  • 角色攀爬系统：设计并实现了高度仿真的攀爬机制，极大增强沉浸感与物理反馈。
  • Socket 组件交互：构建灵活的组件通信与互动逻辑，提升系统模块化和扩展性。
  • PXR_Manager 应用：深入探索无手柄交互的潜力，实践了包括虚拟钢琴演奏、多样手势指令识别和沉浸式射击体验在内的多项创新应用。

案例 2: SteamVR 和 XRToolkit 博物馆项目实践

  • URP（通用渲染管线）：采用 URP 优化项目渲染性能，同时实现高质量的视觉呈现效果。
  • 光照探针与烘焙：通过精细的光照探针和光照烘焙技术，营造真实且富有层次感的展馆光影环境。
  • DoTween 插件：巧妙运用 DoTween 插件，为 UI 元素和场景动画增添流畅自然的过渡效果，提升用户体验。
  • SteamVR 交互：针对 SteamVR 平台深度定制交互系统，实现包括自定义抓取手势和高效的手柄射线检测交互在内的多种互动方式。

案例 3: 敦煌莫高窟博物馆项目

  • 协程函数应用：灵活运用协程函数，实现打字机效果、场景淡入淡出、平滑移动等复杂动画和状态控制。
  • XRTK 深度交互集成：将 XRTK（XR Interaction Toolkit）与项目核心功能深度融合，构建稳定且富有沉浸感的交互体系。
  • PXR 场景切换特效：开发了多样的 PXR 场景切换特效，确保用户在不同场景间切换时拥有流畅且富有艺术感的视觉体验。
  • 拼接游戏模块：设计并实现了互动拼接游戏，精确运用碰撞检测、材质动态切换、严谨的顺序逻辑判断，确保游戏互动的准确性和趣味性。
  • 竞赛样题背景：此项目是为2025 年福建省职业院校技能大赛样题进行的实战演练，旨在模拟比赛环境，在4 小时限时内完成核心功能开发。部分细节尚待进一步优化。
